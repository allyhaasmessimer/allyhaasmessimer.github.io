MySwolePal

MySwolePal is a comprehensive workout planner that offers a range of features to help fitness enthusiasts achieve their fitness goals. With MySwolePal, users can create an account to access its full functionality. Once logged in, users can view their saved workouts, add new exercises, and delete workouts or exercises as needed. To get started, users can search MySwolePal's library of exercises, which is filtered based on muscle group, difficulty level, and exercise type. This helps users find exercises that match their fitness goals. From here, users can add exercises directly to their workout, customize their routine, and make it more effective.

Auto Dealership

The auto dealership application is a comprehensive solution designed to streamline the management of automobile dealerships. With this app, dealership owners and managers can easily manage their inventory, service center, and sales processes in one centralized location. The app consists of three microservices, namely Sales, Inventory, and Services, which are able to interact seamlessly through polling. The backend is built using Django and is containerized using Docker, while the front-end is built using React. Overall, this app is scalable, flexible, and efficient, making it an ideal solution for managing the various aspects of automobile dealerships.


Skills: React.js · Docker · Python

Rate Book by Cover

Open Review Books by Cover is a blog where users can review books solely based on their cover. The application allows users to browse existing reviews and add their own posts with a book's image, title, creator name, content, and delete their own reviews. The blog is designed with API and data models and built using FastAPI, React UI, and SQL Alchemy for database management. The app's main functionalities include viewing a list of blog posts with book images and reviews, deleting a review from the blog, and adding a new post to the open blog.
